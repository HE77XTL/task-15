# task-15
## How to start</br>
Clone this repository to your local folder</br>
Open your terminal, cd to target folder, run</br>
```
cd load-more</br>
node app.js</br>
```
Open your browser, and navigate to http://localhost:8080/index.html</br>
解释代码</br>
__ app.js</br>
__ static</br>
 |__ index.html</br>
app.js 是后端文件，用来创建服务器</br>
static 是静态文件目录</br>
index.html 是静态文件</br>
